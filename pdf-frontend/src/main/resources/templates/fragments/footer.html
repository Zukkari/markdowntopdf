About the project

I decided to make this project in my free time, because all of the existing markdown to pdf converters were either not working for me or too limiting. One of the limitations that I found annoying is that some services accept only up to 1 megabyte files. When you add some images to your markdown files, it is pretty easy to exceed the 1 megabyte limit. So here we are, using my own markdown to pdf converter.
